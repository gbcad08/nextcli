## 跳板机：不直连服务器，安全上了一个台阶

很多人对跳板机不太了解，这里简单解释一下。

跳板机的逻辑是：你的电脑 → 跳板机 → 目标服务器。你不直接暴露目标服务器的IP，所有操作都通过跳板机中转，大大降低了服务器被扫描攻击的概率。

NextCLi 的跳板机系统基于 CentOS 7.8，支持 Windows 远程桌面连接和 Linux 命令行连接，还支持批量自动化运维。套装方案把跳板机和 Windows 工作机打包在一起，两台机器之间走内网互连，延迟极低，外网带宽也不受影响。

对于需要频繁操作多台服务器的运维、开发团队来说，这是个相当实用的方案。
